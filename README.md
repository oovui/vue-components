# vueui
a UI components library based on vue

# Installation
Install the @oovui/vue package:

```sh
npm i -S @oovui/vue 
# or
yarn add  @oovui/vue 
```
Install the Vue plugin:

```javascript
import Vue from 'vue'
import oovui from '@oovui/vue'

Vue.use(oovui)
```

Import the CSS:

```javascript
import '@oovui/vue/dist/oovui.css';
```
